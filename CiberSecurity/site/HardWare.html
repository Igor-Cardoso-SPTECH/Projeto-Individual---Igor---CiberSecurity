Hardware de Hacking: Ferramentas e Aplicações

Raspberry Pi 4

Função: Microcomputador versátil para automação de ataques.

Uso: Execução de Kali Linux, scanners de rede, payloads automatizados.

S.O.: Kali Linux ARM, Raspberry Pi OS.

Especificações: CPU quad-core 1.5GHz, até 8GB RAM, Wi-Fi/BT integrados.

Cenários: Pentest físico e remoto, engenharia reversa e sniffing de pacotes.

Wi-Fi Pineapple Mark VII

Função: Análise e ataque a redes Wi-Fi.

Uso: Rogue AP, MITM, captura de handshakes WPA2.

S.O.: PineAP (Linux modificado).

Alcance: Até 100m com antena direcional.

Cenários: Hotéis, aeroportos, auditoria de segurança Wi-Fi.

Alfa AWUS036ACH

Função: Placa wireless de alta potência para auditoria.

Uso: Modo monitor, injeção de pacotes, deauth attacks.

S.O.: Kali Linux, Parrot OS.

Frequência: Dual-band (2.4GHz/5GHz).

Cenários: Quebra de WPA2, coleta de tráfego criptografado.

LAN Turtle

Função: Backdoor de rede físico e remoto.

Uso: Keylogger, DNS spoofing, shell reverso.

S.O.: Turtle Shell (OpenWRT).

Conexão: Ethernet com tunelamento SSH.

Cenários: Infiltração em redes locais, persistência em ambientes corporativos.

USB Rubber Ducky

Função: Simula teclado para injeção de comandos.

Uso: Execução de scripts maliciosos em segundos.

S.O.: Compatível com qualquer OS com entrada USB.

Velocidade: ~100 teclas/s.

Cenários: Acesso físico não autorizado, engenharia social.

Proxmark3 RDV4

Função: Leitura e clonagem de RFID/NFC.

Uso: Emulação de cartões, leitura à distância, fuzzing de protocolos.

S.O.: Firmware próprio + host via Linux/Windows.

Alcance: ~5cm leitura ativa, passiva maior com antenas especiais.

Cenários: Acesso físico via crachás, clonagem de cartões RFID.

Bash Bunny

Função: Dispositivo USB multi-função de automação.

Uso: Simula teclado, armazenamento, rede; executa payloads.

S.O.: Linux customizado.

Portas: USB 3.0 com múltiplos modos HID/ETH/MSD.

Cenários: Reconhecimento rápido, ataque físico furtivo.

HackRF One

Função: Transceptor SDR de banda larga (1 MHz a 6 GHz).

Uso: Replay de sinais, interceptação de comunicação RF, GSM sniffing.

S.O.: Compatível com GQRX, GNU Radio, SDR#.

Largura: 20 MHz banda ativa.

Cenários: Exploração de sinais sem fio, drones, alarmes, rádio digital.

Flipper Zero

Função: Multi-hacking tool portátil.

Uso: RFID, infravermelho, GPIO, emulação de cartão, Bluetooth.

S.O.: Firmware open-source.

Bateria: ~7 dias em standby.

Cenários: Ataques físicos, testes de engenharia reversa, aprendizado técnico.

Alfa Network AWUS1900

Função: Placa Wi-Fi de alta performance com 4 antenas.

Uso: Captura de tráfego simultâneo 2.4/5 GHz, ataques a WPA3.

S.O.: Kali Linux, ParrotOS.

Taxa: até 1900 Mbps.

Cenários: Testes em redes densas, auditorias corporativas.

O.MG Cable

Função: Cabo USB com hardware oculto para ataques remotos.

Uso: Execução de comandos por Wi-Fi, keylogging, payloads furtivos.

S.O.: Interface web via Wi-Fi + scripts personalizáveis.

Alcance: 50–100m via Wi-Fi integrado.

Cenários: Engenharia social, ataques invisíveis, infiltração física.

USBKill

Função: Dispositivo de destruição física via descarga elétrica.

Uso: Queima de portas USB e circuitos de computadores.

S.O.: Não se aplica (ação puramente elétrica).

Voltagem: Até 220V injetados em milissegundos.

Cenários: Sabotagem, testes extremos de hardware.

USB Nugget

Função: Ferramenta educativa e real para HID injection.

Uso: Execução de scripts HID com tela OLED integrada.

S.O.: CircuitPython + firmware customizado.

Extras: Suporte a payloads customizados e aprendizado interativo.

Cenários: Ensino de cibersegurança, demonstração de vulnerabilidades físicas.
